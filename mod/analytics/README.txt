This Plugin enables Google Analytics for your Elgg website.

After you enable this Plugin, you have to fill in some settings. To do this go to the Administration -> Tools Administration -> Analytics settings.

Here you can configure the following settings:
- Enable tracking: Do you wish to track this site
- Tracking ID: The Google Analytics ID for this site (mostly starts with UA-)
- Enable subdomain tracking: Do you wish to report individual subdomains (see http://www.google.com/support/googleanalytics/bin/answer.py?hl=en&answer=55524 for more information)
- Root domain: For subdomain tracking a Root domain is required

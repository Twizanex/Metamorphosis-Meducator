Version: 1.0

Requires: Tested with the default theme and Elgg 1.6.1

*Description*

This simple plugin adds a customisable link to the 'topbar'. The link appears between the default Tools menu and Settings link and is visible to all users. The link text and URL is configured through the administration screen.

*Admin settings*

Enter the link text and URL.
